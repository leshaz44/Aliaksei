Взят набор данных Heart Disease для предсказания наличия сердечных заболеваний с сайта https://www.kaggle.com/datasets/johnsmith88/heart-disease-dataset
1.	возраст
2.	пол
3.	тип боли в груди (4 значения)
4.	кровяное давление в состоянии покоя
5.	уровень холестерина в сыворотке крови в состоянии покоя, мг/дл
6. уровень сахара в крови натощак > 120 мг/дл
7. результаты электрокардиографии в покое (значения 0,1,2)
8. достигнутая максимальная частота сердечных сокращений
9. стенокардия, вызванная физической нагрузкой
10. олдпик - снижение уровня ST, вызванное физической нагрузкой по сравнению с отдыхом
11. наклон сегмента ST при максимальной нагрузке
12. количество крупных сосудов (0-3), окрашенных при флюороскопии
13. thal: 0 = нормальный; 1 = фиксированный дефект; 2 = обратимый дефект.

Поле "цель" указывает на наличие у пациента сердечно-сосудистых заболеваний. Оно имеет целое значение 0 = отсутствие заболевания и 1 = болезнь.
