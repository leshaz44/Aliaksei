Выбран датасет с Kaggle, включает пять переменных: возраст, опыт работы, должность, уровень образования и заработная плата.
Ссылка: https://www.kaggle.com/datasets/mohithsairamreddy/salary-data
Реализовано 6 методов ML:
- Регрессор Gradient Boosting,
- Экстремальный градиентный бустинг,
- Регрессор CatBoost,
- Регрессия Лассо,
- Байесовская регрессия,
- Линейная регрессия.
Произведена оценка с использованием различных метрик (MAE, MSE, R2), выбран лучший регрессор для данного датасета. Построены различные графики для наглядной оценки эффективности регрессоров.
